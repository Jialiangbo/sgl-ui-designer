关于 SGL

SGL (Small Graphics Library) 是一款专为 MCU 级别处理器设计的轻量级、快速的图形库。它仅需约 3KB RAM 和 15KB ROM 即可运行，支持局部帧缓冲、边界框 + 贪心算法的脏矩形刷新，多种颜色深度（8bit / 16bit / 24bit / 32bit）。


本工具通过可视化拖放的方式帮助嵌入式工程师快速设计 UI 界面，并生成可直接使用的 C 代码。
